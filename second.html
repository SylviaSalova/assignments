LEARNING THE TERMINAL

terminal commands

  1. cd- Used for changing directory
  2. ls- Used for listing items in the folder
  3. mkdir-Used for creating a new folder
  4. rm- Used for deleting a file
  5. rm -rf - Used for deleting a folder
  6. mv- Used for renaming a folder or moving it
  7. touch - Used in creating a new file in a folder
  8. cd ..- Used for moving one step behind to a folder
  9. cd .. /.. - Used for moving two steps behind to a folder
